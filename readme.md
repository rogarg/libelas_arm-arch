## Synopsis

LIBELAS stereo matching library that can be built on an ARM architecture CPU (eg. TX1, TX2, odroid). 

## Installation

- cd /path/to/libelas
- cmake .
- make

Tested on Ubuntu 1404 Nvidia TX1

Check README.txt for information on LIBELAS.
